# NIFTY 50 Stock Price Prediction using LSTM

## Overview
This project predicts NIFTY 50 stock prices using Deep Learning and Time Series Forecasting techniques. Historical stock market data is analyzed and processed using Python libraries, and an LSTM (Long Short-Term Memory) neural network model is implemented to forecast future stock price trends.

The project demonstrates practical applications of Data Science, Machine Learning, and Predictive Analytics in financial market analysis.

---

## Features
- Historical stock market data analysis
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Time series forecasting using LSTM
- Data visualization and trend analysis
- Model evaluation using performance metrics
- Actual vs Predicted stock price comparison

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- TensorFlow / Keras
- Jupyter Notebook
- Yahoo Finance API

---

## Project Workflow
1. Data Collection using Yahoo Finance API
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Scaling and Data Preparation
5. Building the LSTM Model
6. Model Training and Testing
7. Stock Price Prediction
8. Performance Evaluation and Visualization

---

## Machine Learning Model
The project uses an LSTM (Long Short-Term Memory) neural network model, which is highly effective for sequential and time-series forecasting problems.

### Evaluation Metrics
- R² Score
- Mean Absolute Error (MAE)
- Root Mean Square Error (RMSE)
